# 🥻 Vimalavastra – Premium Indian Ethnic Fashion Store

A modern, fully responsive e-commerce website for **Vimalavastra**, a premium Indian saree and ethnic dress brand.

🌐 **Live Site:** [shashi039.github.io/Vimalavastra](https://shashi039.github.io/Vimalavastra/)

---

## ✨ Features

- 🏠 **Homepage** – Hero banner, featured products, categories, testimonials
- 🥻 **Sarees Page** – Product grid with filters (price, fabric)
- 👗 **Dresses Page** – Lehengas, Anarkalis, Salwar suits
- 📋 **Product Detail** – Image gallery, color options, quantity selector, reviews
- 🛒 **Shopping Cart** – Add/remove items, quantity controls, order summary
- 💳 **Checkout** – Address form + UPI / Card / Net Banking / COD payment
- ❤️ **Wishlist** – Save favourite products
- 👤 **Login / Register** – User authentication UI
- ⚙️ **Admin Dashboard** – Manage products, orders, customers, inventory
- ℹ️ **About Us & Contact** – Brand story, WhatsApp, email, social links

## 🚀 How to Update Products

All products are in **`js/products.js`**. To change a price or image:

```js
{ id: 1, name: "Royal Maroon Banarasi Silk Saree", price: 8499, image: "images/saree1.png" }
```

After editing, push to GitHub:

```bash
git add .
git commit -m "Updated products"
git push
```
